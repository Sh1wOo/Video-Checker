# Changelog — Raduga Dataset Video Community

## [0.3.5] — 2025-06-17

### Added
- **Android/VR device support** — подключение Android и VR-гарнитур (PICO 4 Ultra, Quest и др.) через ADB
- **Android File Manager** — полноценный файловый менеджер для Android-устройств:
  - Браузер папок и файлов с двойным кликом по папке
  - Создание, удаление, переименование, перемещение файлов и папок
  - Загрузка файлов на устройство (`adb push`)
  - Скачивание файлов с устройства (`adb pull`)
  - Отображение имени модели устройства (например, PICO 4 Ultra, Galaxy S24)
  - Отображение информации о памяти (`X.X GB свободно из Y GB`)
- **Сканирование Android папок** — `adb pull` + анализ через существующий движок сканера
- **Кастомный NSIS-установщик** с:
  - Кастомным боковым и верхним баннером
  - Страницей лицензии
  - Записью в Add/Remove Programs
  - Определением DLL-зависимостей
  - Проверкой наличия ADB с предложением установить
  - Кастомным Uninstaller с очисткой данных пользователя
- **Лицензия** — Raduga Non-Commercial License v1.0 (бесплатно для некоммерческого использования)
- Добавлен крейт `which` для поиска `adb` в PATH на обеих платформах

### Changed
- Продукт переименован в **Raduga Dataset Video Community** везде (tauri.conf, Cargo.toml, package.json, window title)
- Версия обновлена до **0.3.5**
- Идентификатор: `top.raduga.dataset-video-community`
- Обновлены все команды в `main.rs` с новыми Android IPC-командами

### Fixed
- Название приложения в заголовке окна теперь соответствует бренду
